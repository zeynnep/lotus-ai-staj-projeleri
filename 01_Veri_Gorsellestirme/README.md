# Veri Görselleştirme

Bu klasör, KNIME ve Python kullanarak oluşturduğum veri görselleştirme çalışmalarını içermektedir.

## İçerikler
- Violin Plot (KNIME)
- Bar Chart (KNIME)
- Line Chart (KNIME)
- Pie Chart (KNIME)
- Box Plot (KNIME)
- veri_gorsellestirme.ipynb (Python Notebook)
- Araştırma Dokümanı 

Bu klasörde, veri görselleştirme tekniklerini karşılaştırarak uygun grafik türü seçimi, veri hikayeleştirme ve KNIME workflow tasarımları bulunmaktadır.
