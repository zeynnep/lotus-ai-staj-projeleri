# Sınıflandırma (Classification)

Bu klasör, sınıflandırma problemleri için Python ve KNIME kullanarak geliştirdiğim analizleri içermektedir.

## İçerikler
- KNIME workflow dosyaları (Decision Tree, Random Forest, Logistic Regression)
- Python notebook (classification.ipynb)
- Model değerlendirme metrikleri (Accuracy, Precision, Recall, F1-score)
- CRISP-DM aşama raporları (PDF/DOCX)
- Kullanılan veri setleri

Bu bölümde farklı sınıflandırma algoritmaları karşılaştırılmış ve model performansları analiz edilmiştir.
