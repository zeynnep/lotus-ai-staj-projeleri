# Birliktelik Analizi (Association Rule Mining)

Bu klasör, verisetindeki öğeler arasındaki ilişkileri keşfetmek için kullanılan birliktelik analizi çalışmalarını içermektedir.

## İçerikler
- KNIME workflow dosyaları (Apriori)
- Python notebook (birliktelik_analizi.ipynb)
- Destek (Support), Güven (Confidence) ve Lift hesaplamaları
- Çıkarılan kuralların yorumlanması
- CRISP-DM aşama raporları (PDF/DOCX)
- Kullanılan veri setleri

Bu bölümde Apriori algoritması kullanılarak anlamlı birliktelik kuralları çıkarılmış ve kural metrikleri analiz edilmiştir.
