# Regresyon (Regression)

Bu klasör, regresyon problemleri için Python ve KNIME kullanarak geliştirdiğim analizleri içermektedir.

## İçerikler
- KNIME workflow dosyaları (Linear Regression, Random Forest Regression, KNN Regression)
- Python notebook (regresyon.ipynb)
- Model hata metrikleri: RMSE, MSE, MAE
- CRISP-DM aşama raporları (PDF/DOCX)
- Kullanılan veri setleri

Bu bölümde farklı regresyon algoritmaları karşılaştırılmış ve performans analizleri yapılmıştır.
