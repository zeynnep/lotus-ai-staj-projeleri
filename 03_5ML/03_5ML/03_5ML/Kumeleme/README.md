# Kümeleme (Clustering)

Bu klasör, kümeleme algoritmalarını kullanarak gerçekleştirdiğim analizleri içermektedir.

## İçerikler
- KNIME workflow dosyaları (K-Means, Hierarchical Clustering)
- Python notebook (kumeleme.ipynb)
- Küme sayısı optimizasyon yöntemleri (Elbow, Silhouette Score)
- CRISP-DM aşama raporları (PDF/DOCX)
- Kullanılan veri setleri

Bu bölümde farklı kümeleme teknikleri uygulanmış ve veri yapısına en uygun küme sayısı analiz edilmiştir.
