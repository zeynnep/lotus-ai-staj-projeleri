# 5 Makine Öğrenmesi Alanı (5ML)

Bu klasör, Lotus AI stajı kapsamında geliştirdiğim 5 temel makine öğrenmesi alanına ait tüm analizleri içermektedir:

1. **Sınıflandırma**
2. **Regresyon**
3. **Kümeleme**
4. **Birliktelik Analizi**
5. **Anomali Tespiti**

Her alt klasör içerisinde:
- KNIME workflow dosyaları (.knwf)
- Python Jupyter Notebook dosyaları (.ipynb)
- CRISP-DM aşamalarına ait raporlar (PDF/DOCX)
- Kullanılan veri setleri bulunur.

Bu klasör, makine öğrenmesinin temel tekniklerini uçtan uca uyguladığım kapsamlı bir çalışma yapısını yansıtır.
