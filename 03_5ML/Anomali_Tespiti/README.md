# Anomali Tespiti (Anomaly Detection)

Bu klasör, verideki olağandışı örnekleri tespit etmek için kullanılan anomali tespit yöntemlerine ait çalışmaları içermektedir.

## İçerikler
- KNIME workflow dosyaları (IQR, Z-Score, LOF)
- Python notebook (anomali_tespiti.ipynb)
- Veri ön işleme ve uç değer (outlier) temizleme adımları
- Anomali tespit metrikleri ve görselleştirmeleri
- CRISP-DM aşama raporları (PDF/DOCX)
- Kullanılan veri setleri

Bu bölümde farklı anomali tespit yaklaşımları uygulanmış, anormal gözlemler belirlenmiş ve sonuçlar yorumlanmıştır.
